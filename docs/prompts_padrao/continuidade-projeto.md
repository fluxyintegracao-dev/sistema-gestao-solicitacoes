# Prompt - Continuidade do Projeto

Use este prompt para orientar uma IA nova no repositorio:

```text
Leia AGENTS.md e depois a arvore docs/. Este sistema esta em producao e em uso real. Priorize estabilidade do fluxo principal e do modulo compras. Antes de propor mudancas, identifique runtime ativo, regras de negocio criticas, riscos de deploy e impacto em banco. Nao trate comunicacao interna como modulo isolado; ela ja teve historico de pressao no banco. Explique sempre caminhos de menor risco.
```
