# Documentacao do Projeto

Esta arvore organiza o conhecimento minimo necessario para continuidade segura do sistema por pessoas e por IA.

## Estrutura
- `docs/contexto/`: visao do negocio, estado atual e contexto operacional.
- `docs/arquitetura/`: runtime ativo, backend, frontend, deploy e integracoes.
- `docs/regras_negocio/`: regras criticas de permissao, visibilidade e modulos de negocio.
- `docs/modulos/`: leitura rapida por modulo funcional.
- `docs/logs_desenvolvimento/`: historico recente de manutencao e estabilizacao.
- `docs/prompts_padrao/`: prompts reutilizaveis para continuidade por IA.
- `docs/seguranca/`: checklist operacional e superficie de risco.
